# CSS Challenge

## Objective

The objective of this project is to demonstrate practical understanding of CSS Flexbox, CSS Grid, Responsive Web Design, and CSS Animations. The project consists of small UI components that showcase modern layout techniques and interactive styling.

## Tech Stack

- HTML5
- CSS3

## Features Implemented

### Flexbox Layout
- Three responsive cards displayed in a row on desktop screens
- Cards automatically stack on smaller devices
- Proper alignment and spacing using Flexbox
- Hover interaction effect

### Grid Layout
- Six grid items displayed using CSS Grid
- Responsive column adjustment across screen sizes
- Consistent spacing and alignment

### Animation Component
- Interactive button hover animation
- Smooth transitions
- Professional and subtle motion effects

### Responsive Design
- Mobile-friendly layout
- Flexible card arrangement
- Responsive grid breakpoints
- Readable typography on all devices
